# 포켓몬 도감 (스티커버전)

## 사용 기술

Typescript

React

### 사용 라이브러리

1. motion

```
npm i framer-motion
```

2. react-router-dom

```
npm install react-router-dom
```

3. react-infinite-scroll-hook

```
npm install react-infinite-scroll-hook
```

4. react-intersection-observer-hook

```
npm install react-intersection-observer-hook
```

### 사용 API

1. 포켓몬 API

2. Intersection Observer API
